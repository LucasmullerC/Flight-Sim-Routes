Welcome to Flight Sim Routes, an open source platform designed for aviation enthusiasts and virtual pilots alike. This project mission is to provide a comprehensive solution for generating realistic and customized flights and schedules for Flight Simulators.

  • Flight Sim Routes Random Database is powered by OurAirports.com database, enables the creation of flights based on user preferences or entirely random parameters. For more authenticity, the Real Flights Database, provides access to genuine live flight schedules sourced at moment only from The OpenSky Network and in future more to come.
  • Flight Sim Routes Virtual Airlines Schedules Generator, a groundbreaking feature for virtual airlines. Enables the schedule creation and customization of personalized schedules in a matter of minutes. Compatible and tested with PHPvms 7.x.
